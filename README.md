# Juegos de Beber 🍻
Web interactiva orientada a la gamificación social con sistemas de votación en vivo y salas multijugador.

## 💻 Características Técnicas
- **Frontend:** Motor de plantillas **EJS** para renderizado dinámico en el servidor.
- **Real-time:** `Socket.io` para gestionar las acciones y votaciones de los usuarios al instante.
- **Base de Datos:** MySQL para la gestión de usuarios y estadísticas de juego.

## 🌐 Acceso
Disponible en: [https://juegosbeber.es/](https://juegosbeber.es/)

## 🛠️ Instalación
1. Configura la base de datos en el `.env`.
2. `npm install`
3. `npm start`
